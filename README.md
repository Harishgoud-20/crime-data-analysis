# Crime Analysis
This project was completed as part of a bootcamp capstone assignment. It involves analyzing real-world crime data using Python and MySQL to uncover patterns related to crime hotspots, victim demographics, crime types, and case statuses.

#Dataset
The dataset contains 499 records with the following information:

DR No, Date Reported, Date Occurred
Area Name, Crime Code, Crime Description
Victim Age, Victim Sex
Premises Description, Status
Location, Latitude, Longitude

#Tools & Libraries
MySQL Workbench — Database setup and data import
PyMySQL — Connecting Python to MySQL
Pandas — Data loading and analysis
Matplotlib & Seaborn — Data visualization
Plotly — Interactive crime hotspot map
Jupyter Notebook — Development environment

#Key Findings
Most crimes are concentrated in the Central area
Majority of victims are adults between 20–50 years old
Male victims outnumber female victims in this dataset
Most crimes occur in single family dwellings and streets
Most reported cases are still under Investigation Continued (IC) status
